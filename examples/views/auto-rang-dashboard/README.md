# auto-rang-dashboard

## Props

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|--- | --- | --- | --- | --- |
| value | 绑定值滑动条的值 | number | - | - |
| title | 标题 | string | - | - |
| min | 最小值 | number |	- | - |
| max | 最大值 | number | - | - |
| step | 步长 | number | - | - |
| type | 显示样式 | string | none/both/touch/button | - |
| disabled | 是否禁用 | boolean | - | false |
| unit | 单位 | string | - | - |

## Events
| 事件名称 | 说明 |	回调参数 |
|--- | --- | --- |
| input | 提交修改value | 新的值 |
