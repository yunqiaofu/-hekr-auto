# auto-enum-dropmenu

## Props

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|--- | --- | --- | --- | --- |
| value | 绑定值,选择的下标 | number | - | 0 |
| title | 左侧标题 | string | - | - |
| items | 选择项目列表 | array |	- | - |
| disabled | 是否禁用 | boolean | - | false |

## Events
| 事件名称 | 说明 |	回调参数 |
|--- | --- | --- |
| input | 提交修改value | 新的下标 |
