# auto-bool-switch

## Props

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|--- | --- | --- | --- | --- |
| value | 绑定值 | boolean | - | false |
| title | 左侧标题 | string | - | - |
| switchProps | 开关的属性 | object |	- | - |
| disabled | 是否禁用 | boolean | - | false |

## Events
| 事件名称 | 说明 |	回调参数 |
|--- | --- | --- |
| input | 提交修改value | switch的状态 |
